Random Restaurant & Bar Locator

Collaborative Group Project for UPenn's LPS Coding Boot Camp. This web application uses multiple APIs for locating a nearby restaurant or bar of your choice and displays the location on a map. Businesses can be saved for later if you're feeling it but not right now or passed on if it's not your style. Made with jQuery.

GameSwap: A Video Game Trading App

Full stack web application made with node.js, Express Handlebars, jQuery, MySQL, Dotenv, Bcrypt.js, Sequelize, and several Axios API calls using APIcalypse.
Users create a profile, add games to their library that they own and would like to trade off, can search for games that other users are willing to trade.

Banana Quiz

Timed quiz about everyone's favorite yellow, long fruit! Mobile-friendly web application responsive to several kinds of user input. Right answers are rewarded with a custom prompt, wrong answers receive another prompt, and depending on how well (or poorly) you score, the end result is accompained by a snarky comment. Scores can be saved for tracking your high score or competing against friends. Made with Javascript.
Github repository

Weather Dashboard

Custom weather dashboard with color-coded UV index. Search results pull from a custom weather API to display the current conditions, upcoming 5-day forecast, and stores search history in a sidebar. Sidebar is also stored to local storage to review past search results. Made with Javascript.
